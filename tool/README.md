# tool

## レーティング推定器 (infer_rating.py)

勝数と敗数を入れたら区間を出してほしい。  
厳密な方法で視覚的にわかりやすいのがないので作った。

### 使い方

- 引数は順に勝ち数、負け数、引き分け数

python infer_rating.py 60 40 10
